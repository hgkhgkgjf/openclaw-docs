# 配置示例

下面这些例子可以直接作为 `~/.openclaw/openclaw.json` 的起点。逐字段说明请参阅[配置](/tutorials/gateway/configuration)。

---

## 快速入门

### 最小配置

```json5
{
  agents: { defaults: { workspace: "~/.openclaw/workspace" } },
  channels: { whatsapp: { allowFrom: ["+15555550123"] } },
}
```

保存到 `~/.openclaw/openclaw.json` 后，就可以从该号码向助手发送私信。

### 推荐入门配置

```json5
{
  agents: {
    defaults: {
      workspace: "~/.openclaw/workspace",
      model: { primary: "anthropic/claude-sonnet-4-5" },
    },
    list: [
      {
        id: "main",
        default: true,
        identity: {
          name: "Clawd",
          theme: "helpful assistant",
        },
      },
    ],
  },
  channels: {
    whatsapp: {
      allowFrom: ["+15555550123"],
      groups: { "*": { requireMention: true } },
    },
  },
}
```

## 常见模式

### 多平台设置

```json5
{
  agents: { defaults: { workspace: "~/.openclaw/workspace" } },
  channels: {
    whatsapp: { allowFrom: ["+15555550123"] },
    telegram: {
      enabled: true,
      botToken: "YOUR_TOKEN",
      allowFrom: ["123456789"],
    },
    discord: {
      enabled: true,
      token: "YOUR_TOKEN",
      dm: { allowFrom: ["yourname"] },
    },
  },
}
```

### 安全私信模式（共享收件箱/多用户私信）

如果多人可以向你的助手发送私信（`allowFrom` 中有多个条目、为多人配对审批、或 `dmPolicy: "open"`），建议启用安全私信模式，让不同发送者默认使用各自的上下文：

```json5
{
  // 安全私信模式（推荐用于多用户或敏感私信智能体（Agent））
  session: { dmScope: "per-channel-peer" },

  channels: {
    // 示例：WhatsApp 多用户收件箱
    whatsapp: {
      dmPolicy: "allowlist",
      allowFrom: ["+15555550123", "+15555550124"],
    },

    // 示例：Discord 多用户收件箱
    discord: {
      enabled: true,
      token: "YOUR_DISCORD_BOT_TOKEN",
      dm: { enabled: true, allowFrom: ["alice", "bob"] },
    },
  },
}
```

### OAuth + API 密钥回退

```json5
{
  auth: {
    profiles: {
      "anthropic:subscription": {
        provider: "anthropic",
        mode: "oauth",
        email: "me@example.com",
      },
      "anthropic:api": {
        provider: "anthropic",
        mode: "api_key",
      },
    },
    order: {
      anthropic: ["anthropic:subscription", "anthropic:api"],
    },
  },
  agents: {
    defaults: {
      workspace: "~/.openclaw/workspace",
      model: {
        primary: "anthropic/claude-sonnet-4-5",
        fallbacks: ["anthropic/claude-opus-4-6"],
      },
    },
  },
}
```

## 提示

- 如果你设置 `dmPolicy: "open"`，对应的 `allowFrom` 列表必须包含 `"*"`。
- 提供商（Provider）ID 不同（电话号码、用户 ID、频道 ID）。使用提供商（Provider）文档确认格式。
- 可选的部分可以稍后添加：`web`、`browser`、`ui`、`discovery`、`canvasHost`、`talk`、`signal`、`imessage`。
- 更深入的设置说明请参阅[提供商（Provider）](/tutorials/channels/whatsapp)和[故障排查](/tutorials/gateway/troubleshooting)。
